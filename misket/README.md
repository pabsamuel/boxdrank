# Misket — Turkish Marbles 🎯

A single-file browser game of the classic Turkish street marbles game (*misket / bilye*), circle variant: flick your shooter (*enek*) to knock marbles out of the chalk circle — everything you knock out is yours. Human vs. AI, best of 3 rounds, with an optional **Kanlı (for keeps)** mode. Vanilla JS + Canvas 2D, zero dependencies, works offline, plays with mouse or touch.

**Play locally:** just double-click `index.html`.

## Deploy

**GitHub Pages:** push this folder to a repository, then go to *Settings → Pages*, choose *Deploy from a branch*, pick your branch and the folder containing `index.html` (root or `/docs`), and save — your game will be live at `https://<user>.github.io/<repo>/` within a minute. If `index.html` lives in a subfolder like `misket/`, either move it to the root/`docs` or open `https://<user>.github.io/<repo>/misket/`.

**Netlify Drop:** open [app.netlify.com/drop](https://app.netlify.com/drop) and drag this folder onto the page — Netlify uploads it and gives you a live URL instantly, no account configuration or build settings needed. (Vercel and Cloudflare Pages work the same way: import the folder as a static site with no build command.)
